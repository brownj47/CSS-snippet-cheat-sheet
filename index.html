CSS Snippet Cheat Sheet

Below are some useful CSS snippets, condensed for easy copy and paste!


Flexbox Row

Use these properties to create a Flexbox Row Layout.


.row {
    display: flex;
    flex-direction: row;
    flex-wrap:wrap;
}

Flexbox Column

Create a flexbox in column layout.


.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

A 12 column layout using CSS grid.


.grid {
    display: grid;
    width: 100%;
    grid-template-columns: 100%:
    repeat(12, 1fr);
}

Linear Gradients

A linear background gradient from top to bottom.


.linear-gradient-background {
    background-image: linear-gradient
    (red 0%, white 100%);
}

Box Transition Glow

Use transition and box-shadows to glow on hover.


.tile .tile-title {
    border-radius: 8px;
    transition: all 0.5s ease-in
    -out;
}

.tile:hover,
 .tile:hover .tile-title {
    inset: 0 0 8px white;
}
.

Overlay with Card Title

Use position properties to raise an element higher than the natural starting point.


.tile-title {
    position: relative;
    margin-top: -20px;
}
